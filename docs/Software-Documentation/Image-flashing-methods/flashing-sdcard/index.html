Flashing image in SD card

Besides eMMC, the image can also be booted from SD card( in SD card mode). In order to do that, the SD card must have a bootable SD card image.

Creating a bootable SD card image​

An existing image(SYNAIMG/eMMCIMG) can be converted into a bootable SD card image. Follow the below steps to create a bootable SD card image :

  1. Download gen_sd.sh script from Github and paste it in the image folder (SYNAIMG/eMMCIMG). The script runs in a Linux environment with the mkfs.ext4, gzip, gdisk, and sgdisk utilties installed.

  2. Make the file executable and run it from within the image folder

chmod a+x gen_sd.sh
./gen_sd.sh
  1. If the script executes successfully as shown below, an output file SD.img is created. This is the bootable image to be flashed in the SD card.

  1. Flash the bootable image into the SD card using balena etcher tool.

When balena etcher is open, select the SD.img created inside the image folder.

After inserting the SD card into the host PC, select the SD card from the storage option.

Then click the flash option. The image will get flashed. If flashing is successful, a success message will be displayed as shown.

note

It may take about 10 - 20 minutes to flash the image into the SD card. This time depends on the speed of SD card and the host machine.

After flashing, insert the SD card into the SD slot as shown below and boot the device in SD card mode.
